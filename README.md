\# 🧠 EEG Eye State Detection



A Machine Learning-based web application that detects 

eye state (Open / Closed) from 14-channel EEG signals 

using the standard 10-20 electrode placement system.



\## 🚀 Tech Stack

\- Python 3.14

\- Django 6.0

\- Scikit-learn

\- Pandas \& NumPy

\- Bootstrap 5

\- SQLite



\## 📊 ML Models \& Accuracy

| Model | Accuracy |

|-------|----------|

| 🏆 Random Forest | 93.22% |

| KNN | 91.02% |

| Decision Tree | 82.24% |

| SVM | 69.39% |

| Logistic Regression | 58.51% |

| Naive Bayes | 45.59% |



\## ⚙️ How to Run

1\. Clone the repository

2\. Create virtual environment: `python -m venv venv`

3\. Activate: `venv\\Scripts\\activate`

4\. Install packages: `pip install -r requirements.txt`

5\. Train models: `python train\_models.py`

6\. Run server: `python manage.py runserver`

7\. Open: `http://127.0.0.1:8000/`



\## 🎯 Features

\- User Registration \& Login

\- 14 EEG Channel Input Form

\- Predicts eye state using 6 ML models

\- Model comparison table

\- Best model highlighted in green



\## 🧠 EEG Channels Used
